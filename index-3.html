Como editar este calendário

Altere as informações nos campos abaixo (períodos, feriados, recessos, sábados letivos, avaliações e eventos). A cada alteração, clique em Gerar calendário para atualizar a visualização acima. Use Gerar PDF / Imprimir para divulgar aos servidores, e Salvar dados (arquivo) para guardar suas alterações — depois é só Abrir dados (arquivo) em outro computador ou no próximo ano para continuar de onde parou.

1. Dados gerais

Identificação que aparecerá no topo do calendário gerado.

2. Períodos letivos

Defina o início e o fim de cada semestre/etapa. Todos os dias de segunda a sexta dentro desses intervalos viram automaticamente dias letivos — exceto os que você marcar como feriado, recesso ou férias na etapa seguinte.

3. Exceções e eventos

Adicione feriados, recessos, férias, sábados letivos, avaliações e planejamentos. Use data fim para intervalos (ex: férias de 02/01 a 21/01). Deixe a data fim vazia para um único dia.

"Gerar calendário" atualiza a visualização no topo da página. "Baixar PDF" abre a janela de impressão — escolha Salvar como PDF como destino para gerar o arquivo, ou imprima direto.
